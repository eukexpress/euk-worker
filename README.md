# EukExpress Global Logistics

A professional global logistics management platform with single-admin architecture.

## Features

- Complete shipment lifecycle management
- Three-layer admin interface
- Intelligent status tracking
- Intervention controls (Customs, Security, Damage, Return, Delay)
- Automated email communications
- Public tracking capabilities
- QR code generation
- Invoice PDF generation

## Technology Stack

- **Backend**: FastAPI (Python 3.10+)
- **Database**: PostgreSQL 14+
- **Frontend**: HTML/CSS/JavaScript with Alpine.js
- **Email**: Resend API
- **Authentication**: JWT + OAuth2

## Quick Start

1. Clone this repository
2. Set up Python virtual environment
3. Install dependencies
4. Configure environment variables
5. Run database migrations
6. Start the development server
